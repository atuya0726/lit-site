# lit. Webサイト要件整理と実装メモ

## 1. いただいた要件

- 栃木県のソフトテニスクラブチーム `Lit.` のWebサイトを作成する
- ページ構成は以下を想定
  - ホーム
  - 練習について
  - 試合結果
  - 問い合わせ
- 問い合わせは Googleフォーム埋め込みで対応する
- 試合結果は `json` / `md` などの構造化データで管理し、ひとことコメントを添えられるようにする
- 配色は原則2色
  - プライマリー: `#623079`
  - セカンダリー: `#F8F9FA`
- デザインの方向性
  - ださくない / コーポレート寄り / モダン
  - 色のルールと必要な情報さえ守れば手法は自由（グラスモーフィズム / モダン等OK）

## 2. 実装した内容

### デザインシステム

- フォント
  - 表示用 / 英文: `Inter`（ウェイト差で階層を作る）
  - 和文: `Noto Sans JP`
  - ※当初は `Fraunces` を表示用に使っていたが、トーン調整のため `Inter` に統一
- カラートークン（CSS変数）
  - `--primary` `#623079`
  - `--primary-deep` `#2a1235`（プライマリーの濃色派生）
  - `--secondary` `#f8f9fa`
  - `--ink` / `--ink-soft`（本文用の墨色）
- ビジュアル言語
  - 背景に大きなパープル系オーブを浮かせ、グリッド線をマスクで重ねた抽象背景
  - 主要ブロックはガラスモーフィズム（`backdrop-filter` + 半透明）
  - アクセントは濃いパープルのダークカード
  - 角丸 24–28px / 柔らかい大きな影で立体感

### 共通レイアウト (`src/layouts/Layout.astro`)

- Google Fonts 読み込み
- 背景オーブ + マスクされたグリッド
- フローティングのガラスナビ（英語ラベル + 日本語サブラベル）
- 上品なフッター（ブランド・ナビ・コピーの3カラム）

### ページ実装

- `src/pages/index.astro`
  - 大型ヒロー（巨大タイポ + 斜体ミックス）
  - ガラスカード内に統計（活動頻度 / 出場大会数 / レベル）
  - サイドにダーク調の「最新試合結果」カード
  - 下段は3セルのベントグリッド
- `src/pages/practice.astro`
  - エディトリアル風ページヘッダー
  - スケジュール / エリア / 対象 / 持ち物のベントグリッド
  - 持ち物はチップ表示
- `src/pages/results.astro`
  - 最新の試合結果を大型のフィーチャーカードで強調
  - 残りはアーカイブ一覧（日付バッジ付き）
- `src/pages/contact.astro`
  - 左にダークなインフォカード（返信目安・エリア・手段）
  - 右にガラスのフォームフレーム + Googleフォーム埋め込み

### 試合結果データ

- `src/data/match-results.json`
  - `id`, `date`, `tournament`, `category`, `result`, `comment`
  - `results` ページで日付降順に表示、トップとresultsの両方で活用

## 3. 補完した内容（提案を含む）

- ナビ・フッターを含めた全ページ共通の体験
- レスポンシブ対応（768〜960px 周辺で1カラムに切替）
- 演出
  - 背景オーブのアニメーション（ゆっくりドリフト）
  - カードのホバー浮き上がり
- Googleフォーム
  - `src/pages/contact.astro` の `googleFormEmbedUrl` を実URLに差し替えるだけで運用可
- データ運用
  - 試合結果は JSON 管理
  - 記事的な運用に拡張する場合は Astro Content Collections + Markdown へ移行可能
